# DSA Visualizer

A dynamic and interactive web application to visualize and understand popular Data Structures and Algorithms (DSA) such as Sorting and Searching.  
This project is built with **HTML**, **CSS**, and **JavaScript**—no frameworks required!

---

## ✨ Features

- **Modern, Attractive UI:** Clean, responsive design with animated gradients and interactive controls.
- **Sorting Visualizer:**  
  - Supports Bubble, Selection, Insertion, Quick, and Merge Sort.
  - Step-by-step animation with Pause, Resume, and Stop controls.
  - Input your own array or generate a random one.
- **Searching Visualizer:**  
  - Supports Linear and Binary Search.
  - Step-by-step animation with Pause, Resume, and Stop controls.
  - Input your own array or generate a random one.
- **Centralized Styling:**  
  - All styles are managed in `style.css` for consistency and easy customization.

---

## 🚀 Getting Started

1. **Clone or Download** this repository.
2. Open the project folder in your code editor.
3. Open `index.html` in your browser to launch the homepage.

---

## 📁 Project Structure

```
dsa-visualizer/
│
├── index.html         # Homepage (choose Sorting or Searching)
├── sorting.html       # Sorting visualizer page
├── searching.html     # Searching visualizer page
├── style.css          # Centralized styles for all pages
└── script.js          # (Optional) For shared JS logic if needed
```

---

## 🖥️ Usage

- **Homepage:**  
  Select "Sorting" or "Searching" and click **Visualize**.
- **Sorting/Searching Pages:**  
  - Choose the algorithm.
  - Generate a random array or enter your own.
  - Enter the element to search (for searching).
  - Use **Run**, **Pause**, **Resume**, and **Stop** to control the animation.

---

## 🎨 Customization

- **Colors & Styles:**  
  Edit `style.css` to change color schemes, fonts, or layout.
- **Algorithms:**  
  Add or modify algorithms in the respective HTML files’ `<script>` sections.
